## Getting Started

1. Clone the repository to your local machine: 
```bash
git clone https://github.com/tanovikova22/ColorPicker.git
```
2. Install dependencies
```bash
yarn install
```
3. Run the code: 
```bash
yarn start
```

### Prerequisites

```bash
node v12.13.1
npm v6.14.8 
```
